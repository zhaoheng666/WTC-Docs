## 直接复制到 Gamma.app 的提示词

```
创建一份 8 页的专业 PPT，介绍 WTC-Docs 文档系统。

风格要求：
- 科技感、简洁现代
- 主色调：深蓝色 (#1a3a6c) + 金色强调 (#d4a84b)
- 适合技术团队内部分享

页面结构：

第1页 - 封面
标题：WTC-Docs
副标题：WorldTourCasino 项目专属文档系统
口号：人员友好 · AI 友好
副口号：让项目知识积累、传承更容易，让 AI 上下文补充更高效

第2页 - 双重定位
标题：双重定位：人员友好 + AI 友好
左侧卡片 - 人员友好：
• 智能中文搜索 - 快速定位文档
• 自动侧边栏 - 目录结构一目了然
• 飞书一键转换 - 无需手动整理格式
• 位置无关设计 - 文档可自由移动

右侧卡片 - AI 友好：
• AI 上下文知识库 - 补充 CLAUDE.md
• 结构化 Markdown - 便于 AI 解析理解
• 项目专属规则 - 可迭代积累规范
• 文档即代码 - Git 版本控制

第3页 - 解决的问题
标题：项目知识积累与传承
左侧（红色/问题）：
❌ 传统方式的痛点
• 文档分散在飞书、本地、聊天记录
• 关键经验只存在于个人记忆中
• 新成员上手困难，反复踩坑
• 问题解决方案无法复用

右侧（绿色/解决）：
✅ WTC-Docs 解决方案
• 统一文档中心，集中管理
• 经验沉淀为结构化文档
• 智能搜索快速定位
• 故障排查案例库复用

底部：文档分类 - 活动 | 关卡 | Native | 工程-工具 | 服务器 | 故障排查 | 成员

第4页 - AI 上下文知识库
标题：AI 上下文知识库
副标题：补充 CLAUDE.md，构建项目专属 AI 知识体系

三列布局：
列1 - CLAUDE.md 核心上下文：
• 项目架构概述
• 代码规范规则
• Git 提交流程
• 禁止与强制行为

列2 - docs/ 文档库扩展知识：
• 详细技术规范
• 故障排查案例
• 活动/关卡配置
• 工具使用指南

列3 - 持续迭代知识积累：
• 每次问题解决沉淀
• AI 规则持续优化
• 项目知识不断丰富
• 上下文越用越精准

底部高亮：核心价值 - AI 辅助开发时，自动获取项目专属知识，无需重复解释项目背景

第5页 - 独创设计
标题：独创设计：位置无关链接
副标题：文档可自由移动，链接永不失效

对比展示：
❌ 传统相对路径：../assets/image.png → 移动文件链接失效
✅ 根路径设计：/assets/screenshot.png → 移动文件链接有效

核心优势列表：
• 文档重组自由：可随时调整目录结构，无需修改任何链接
• 图片统一管理：所有图片存储在 /assets/，MD5 哈希命名，自动去重
• 维护成本低：无需担心移动文件导致的死链问题

第6页 - 自动化流水线
标题：6 阶段全自动化构建流水线
副标题：npm run build 一键完成，无需人工干预

流程图或表格：
1. fix-filenames.js → 移除非法字符、处理长文件名
2. image-processor.js → 下载外部图片、去重、清理未引用
3. pdf-processor.js → 收集处理 PDF 资源
4. link-processor.js → 相对路径转绝对路径
5. generate-directory-index.js → 自动生成目录索引
6. vitepress build → 最终构建输出 dist/

第7页 - 飞书集成
标题：飞书文档无缝集成
副标题：lark-mcp 工具：一键转换飞书 Wiki 到 Markdown

流程：飞书 Wiki 链接 → lark-mcp 解析 → 生成 MD 文档 → 自动提交

核心功能：
• 自动识别文档类型（故障排查/活动/关卡/其他）
• 自动下载关键截图（1-3张）
• 生成规范摘要表格
• 保留原文档链接可追溯

使用方式：
提供飞书链接 + "从飞书转 docs" → 自动完成全部流程

第8页 - 总结
标题：WTC-Docs 核心价值总结

四个价值卡片：
👥 人员友好 - 智能搜索 + 自动侧边栏 + 飞书集成
🤖 AI 友好 - CLAUDE.md 补充 + 规则迭代积累
📚 知识传承 - 经验沉淀 + 案例复用 + 版本追踪
⚡ 全自动化 - 6 阶段流水线 + Git 推送即部署

底部口号：让项目知识积累、传承更容易，让 AI 上下文补充更高效
线上地址：https://zhaoheng666.github.io/WTC-Docs/
```

## 使用步骤

1. 打开 https://gamma.app（免费注册）
2. 点击 "Create new" → "Paste in text"
3. 复制上面的提示词内容
4. 选择风格（推荐：Professional / Tech）
5. 生成后微调
6. 导出为 PPTX
