# DH 机台入口 - 同台加载未适配占位入口问题

## 摘要

|          |                                    |
| -------- | ---------------------------------- |
| **状态**   | ✅ 已解决                              |
| **日期**   | 2026年4月10日                         |
| **原因**   | DH 机台入口未适配同台加载，导致新界面始终展示占位入口 |
| **涉及人员** | 陈天琦、马新蓝                            |

---

## 问题

DH 机台入口在同台加载场景下未做适配，导致资源未加载出来时，新界面始终展示占位入口（placeholder）。

**问题截图：**

![DH机台入口展示占位入口](/assets/10dabbaed6bb81bffe2e80629757dc44.png)

*没加载出资源的时候，一直就这样展示占位入口*

**触发场景**：
- DH 机台使用同台加载方式进入
- 资源尚未加载完成时，新界面展示占位入口而非正常入口

---

## 解决方案

对 DH 机台入口逻辑进行同台加载适配，确保资源加载完成后再显示真实入口，避免占位入口持续展示。

---

## 测试

- ✅ 问题已解决，已验证通过
